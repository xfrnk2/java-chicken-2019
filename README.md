- [x] 메인 화면을 출력하는 기능

    ```
    [예외]

    ```
- [ ] 테이블 목록을 출력하는 기능
- [ ] 메뉴를 출력하는 기능
- [ ] 메인 화면에서 "1" 선택시, 테이블과 등록할 메뉴, 수량을 입력받고 주문을 등록하는 기능
    ```
    [예외]
    - 없는 테이블이면 예외처리
    - 없는 메뉴이면 예외처리
    - 수량이 숫자가 아니면 예외처리
    - 수량이 0보다 작으면 예외처리
    ```
- [ ] 메인 화면에서 "2" 선택시, 테이블과 결제 유형을 입력 받고 결제를 진행하는 기능
    ```
    [예외]
    - 없는 테이블이면 예외처리
    - 없는 결제 유형이면 예외처리 (현존 결제 유형은 1, 2)
    - 할인율을 적용한 최종 결제 금액이 정수가 아닌 소수인 경우 예외처리
    ```
- [ ] 최종 결제할 금액을 출력하는 기능
- [ ] 메인 화면에서 "3" 선택시, 프로그램을 종료하는 기능

# 치킨집 POS 미션 구현해보기

## 기능 요구 사항
- 치킨집 사장님이 사용하는 간단한 포스(POS) 프로그램을 구현한다.  
  **주문 등록, 결제하기, 프로그램 종료 기능을 가진다.**

- 메뉴 기본 정보가 주어지며 **메뉴 번호, 종류, 이름, 가격**을 가진다.

- 테이블 기본 정보가 주어지며 **테이블 번호**를 가진다.

- 한 테이블에서 주문할 수 있는 **한 메뉴의 최대 수량은 99개**이다.

- 주문이 등록된 테이블은 **결제가 이루어지기 전까지 테이블 목록에 별도로 표시한다.**

- 주문 내역에 대한 계산을 할 때는 **결제 유형에 따라 할인율**이 달라진다.
    - 치킨 종류 메뉴의 수량 합이 **10개가 넘 경우, 10,000원씩 할인된다.**
        - e.g. 10개는 10,000원 할인, 20개는 20,000원 할인
    - **현금 결제는 5%가 할인**되며 **할인된 금액에서 한 번 더 할인**이 가능하다.

- 주문 혹은 결제가 불가능한 경우 그 이유를 보여 주고, 다시 주문 혹은 결제가 가능하도록 해야 한다.

- 최종 결제 금액을 보여준다.
